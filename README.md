# ubuntu-vscode-c-dev
Vagrant file for GUI C/C++ dev IDE. Ubuntu 18.04, git, buildessentials, vscode with c/cpp tools, and docker.

Requirements:
- VirtualBox 6
- Vagrant


```bash
$ git clone https://github.com/erniebilling/ubuntu-vscode-c-dev.git
$ cd ubuntu-vscode-c-dev
$ vagrant up
```
